Machine Learning

Our Approach

We applied the Random Forest Classifier Algorithm to our data and visualized the results using a Confusion Matrix. Predicted values are on the x-axis while true values appear on the y-axis.

Preprocessing/Pandas

We utilized Sklearn preprocessing in conjunction with Pandas to filter out the six play types, bucketize the yards to go and and concat(get_dummies) to create dummy columns for our analysis.

Sklearn Metrics

Utilizing the Sklearn modules robust metric capabiltiies, we generated an accuracy score, explained variance, classification table and, ultimately, the below confusion matrix.
